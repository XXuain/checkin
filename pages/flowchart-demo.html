流程圖 Demo（README 同款）

使用 Mermaid 在瀏覽器繪製，語法與 README 的 graph TD 區塊相同。

為什麼用 Mermaid？

  • 與文件同源：圖用文字描述，可版本控制、Review 時 diff 清楚。
  • 支援菱形決策 { }、合流 A & B --> C 等常見流程圖。
  • 替代方案：要可拖曳節點、甘特圖級互動可評估 @xyflow/react（React Flow）、AntV G6；單純「規格圖＋文件」Mermaid 通常最省事。
